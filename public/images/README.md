## Görsel Boyutları

- **Hero Slider**: 1920x1080px (landscape)
- **Page Heroes**: 1920x800px (wide landscape)
- **Project Cards**: 800x600px (4:3 ratio)
- **Team Photos**: 400x500px (portrait)
- **Partner Logos**: SVG (scalable) veya 200x100px PNG
- **CTA Background**: 1920x800px

## Görsel Optimizasyonu

Görselleri eklemeden önce:
1. WebP formatına dönüştürün (daha iyi performans)
2. Sıkıştırın (TinyPNG, ImageOptim vb.)
3. Doğru boyutlarda kaydedin
4. Alt text için anlamlı dosya isimleri kullanın

## Placeholder Görseller

Geliştirme aşamasında placeholder görseller için:
- [Unsplash](https://unsplash.com/) - Ücretsiz yüksek kalite fotoğraflar
- [Pexels](https://pexels.com/) - Ücretsiz stok fotoğraflar
- [Placeholder.com](https://placeholder.com/) - Hızlı placeholder'lar
